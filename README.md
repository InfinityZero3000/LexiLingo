## Overview

LexiLingo is a modern language learning application that combines AI tutoring with personalized vocabulary management. Built with Flutter, the app provides an intuitive learning experience across iOS, Android, and Web platforms.

---

## Key Features

- **AI Chat Tutor**: Practice conversations with Google Gemini AI
- **Vocabulary Library**: Personal word collection with search and tracking
- **Structured Courses**: Multiple levels from Beginner to Advanced
- **Progress Tracking**: Monitor your learning journey
- **Cross-Platform**: Available on iOS, Android, and Web

---

## Technology Stack

- **Framework**: Flutter 3.24.0
- **Language**: Dart 3.8.1
- **Architecture**: Clean Architecture
- **AI**: Google Gemini API
- **Auth**: Google Sign-In
- **Database**: SQLite

## Getting Started

### Prerequisites
```bash
Flutter SDK: 3.24.0+
Dart SDK: 3.8.1+
iOS: 13.0+ | Android: API 24+
```

### Installation

1. Clone the repository
```bash
git clone https://github.com/InfinityZero3000/LexiLingo.git
cd LexiLingo/lexilingo_app
```

2. Install dependencies
```bash
flutter pub get
```

3. Run the app
```bash
flutter run
```
