# Sections

This file defines all sections, their ordering, impact levels, and descriptions.
The section ID (in parentheses) is the filename prefix used to group rules.

---

## 1. Spaced Repetition (srs)

**Impact:** CRITICAL  
**Description:** Algorithms and strategies for optimal long-term memory retention through scientifically-proven spaced repetition techniques.

## 2. Content Generation (content)

**Impact:** HIGH  
**Description:** Best practices for generating high-quality, contextually relevant learning content with appropriate difficulty levels and native examples.

## 3. Progress Tracking (progress)

**Impact:** HIGH  
**Description:** Strategies for tracking learner progress, maintaining engagement, and providing meaningful feedback on learning journey.

## 4. Adaptive Learning (adaptive)

**Impact:** HIGH  
**Description:** Techniques for personalizing learning experience based on individual performance, pace, and weak points.

## 5. Pronunciation (pronunciation)

**Impact:** MEDIUM  
**Description:** Methods for analyzing, scoring, and providing feedback on pronunciation accuracy using speech processing technology.

## 6. Gamification (gamification)

**Impact:** MEDIUM  
**Description:** Game design patterns that increase motivation, engagement, and consistency in language learning practice.

## 7. Accessibility (accessibility)

**Impact:** MEDIUM  
**Description:** Ensuring language learning features are accessible to users with diverse abilities and learning needs.
